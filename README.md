# LifeLine Blood Donor Network

A web application that connects blood donors with people who need blood during emergencies.

## Features

• User Registration and Login  
• Role Based Access (User / Admin)  
• Blood Donor Registration  
• Search Blood Donors by City and Blood Group  
• Emergency Blood Request System  
• Google Maps Directions to Donor Location  
• Admin Dashboard for Managing Donors and Requests  
• Duplicate Account Prevention  
• Email and Phone Validation  

## Technologies Used

HTML  
CSS  
JavaScript  
LocalStorage (Browser Database)

## System Roles

### User
- Register account
- Login
- Register as blood donor
- Search for donors
- Submit emergency blood requests

### Admin
- View all donors
- View emergency requests
- Delete invalid donors
- Delete spam requests
- Reset system database

## How to Run

1. Download the project
2. Open `index.html` in browser
3. Register a new user or login
